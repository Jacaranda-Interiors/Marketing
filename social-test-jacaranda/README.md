# Jacaranda Interiors Social Mockups

These files are editable HTML/CSS mockup designs for review. They present four polished Instagram portrait post concepts for **Jacaranda Interiors** using a refined, neutral luxury visual direction.

## Files included

- `index.html` — A simple start page with links to all four mockup posts.
- `styles.css` — Shared brand colors, typography, spacing, faux image styles, and reusable layout classes.
- `post1.html` — Editorial quote post mockup.
- `post2.html` — Split-layout post with placeholder image panels and text.
- `post3.html` — Product spotlight post with luxury styling and material swatches.
- `post4.html` — Soft call-to-action post with a warm interior mood layout.
- `README.md` — This guide.

## How to open the mockups in a browser

### Locally

1. Open the `social-test-jacaranda` folder on your computer.
2. Double-click `index.html`, or right-click it and choose **Open With** your browser.
3. From the index page, click any post link to view the standalone mockup.
4. You can also open `post1.html`, `post2.html`, `post3.html`, or `post4.html` directly.

### In GitHub

GitHub will display the HTML source files, but it will not render them as live web pages in the repository view.

To preview them from GitHub, you can:

1. Download the repository ZIP and open the HTML files locally.
2. Enable GitHub Pages for the repo or branch, then browse to the published site.
3. Use a local editor such as VS Code with a simple preview extension or local server.

## Where to edit text, colors, and layout

### Text content

- Edit headlines, subtitles, captions, and button text directly inside each `post*.html` file.
- Each HTML file includes comments explaining which sections are safest to update.

### Colors and typography

- Open `styles.css`.
- Update the CSS variables near the top inside the `:root` block to change the palette quickly.
- Serif headline and sans-serif body font stacks are also defined near the top of `styles.css`.

### Layout and visual styling

- Shared spacing, borders, shadows, and reusable blocks are all in `styles.css`.
- Faux image placeholders use gradients and layered shapes so the designs remain presentation-ready without photography.
- If you later want to replace placeholders with real images, the easiest path is to swap the placeholder block content or use background images in the same class names.

## Notes

- These mockups are sized to the Instagram portrait proportion of **1080 × 1350**.
- The designs are intentionally minimal, warm, and high-end for presentation and review.
